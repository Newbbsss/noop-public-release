# Ugly — living definition (NOOP)

Refresh this whenever a ship feels “AI product” again.

## Ugly means
- Bolted chrome: extra pills, scrubber bars, floating badges that weren’t there before
- Nested cards (GlowCard inside GlowCard / card on calendar on card)
- Radial glow orbs, glass strips, explanatory chrome that restates the UI
- Invented clinical vitals or fake WHOOP certainty
- Three competing labels for one control

## Avoid by
- Enhance the control that already exists (title hold-swipe, not a new scrubber)
- Flat on sky for calendars and phase reads
- One job per section; one accent at a time
- Interaction containers only when the user is acting (log chips, day panel)
- If removing border/shadow/radius doesn’t hurt understanding — remove it
